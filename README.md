A simple and interactive React app that shows real-time weather updates for different cities.  
It uses the Open-Meteo API to fetch temperature, humidity, and wind speed and visualizes them with beautiful charts using Recharts.

Features:
- Real-time weather data (Delhi, Mumbai, Bengaluru, Kolkata)
- Line, Bar and Pie charts for better visualization
- Responsive layout with clean design
- One-click refresh to update data

Tech Stack:
- React.js
- Recharts for charts
- Open-Meteo API for live weather data
- CSS Grid and Flexbox for layout

Run Locally:
npm install  
npm run dev  
Then open http://localhost:5173 in your browser.
